# database_pathfinding

Using databases to speed up pathfinding

In order to run the project, add the following folders to your in root directory:

- databases
  - databases
    - cover
    - HC
    - DBA
- images
- results
- dynamic
  - databases
    - DBA

Also, add the [Apache Commons IO](https://commons.apache.org/proper/commons-io/download_io.cgi) and 
[Apache Commons Text](https://commons.apache.org/proper/commons-text/download_text.cgi) libraries to your project.
